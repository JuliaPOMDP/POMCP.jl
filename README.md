## Installation

Using POMDPs.jl (should automatically take care of dependencies)

```julia
Pkg.add("POMDPs")
import POMDPs
POMDPs.add("POMCP")
```

OR (some optional dependencies may be missing)

```julia
Pkg.clone("https://github.com/juliapomdp/POMCP.jl.git")
```
